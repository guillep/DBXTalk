Code Repository

The code of the dbxtalk project is hosted in SmalltalkHub, under the DBXTalk group. If you would like to have write access to the repository, do not hesitate to contact us.

Github? We are also planning to migrate soon our source code to GitHub. If you are willing to help us migrate, please leave an email!

Jenkins CI

The projects under the dbxtalk umbrella are automatically tested every day by a Jenkins continuous integration service. We thank inria for the provided service. If you would like to have write access to the ci infrastructure, do not hesitate to contact us.

Github? We are also planning to migrate soon our source code to GitHub. If you are willing to help us migrate, please leave an email!

Issue Tracker

Bugs, issues, enhancements can be posted in our issue tracker.

Backlog & Wishlist

We have a public trello board to keep the list of tasks and the backlog of the project. Do not hesistate to contact us to have write access to the backlog.
