Overview

A GLMRubTextFieldBrick is basic text editor brick used for single line input fields. Supports variety of layout properties:

vShrinkWrap - height fits text size + vertical paddings hShrinkWrap - width fits text size + horizontal paddings vSpaceFill - fills parent vertically. Actual text editor height = parent height - vertical paddings hSpaceFill - fills parent hprozontally. Actual text editor width = parent width - horizontal paddings beCentered - actuall text editor field will be centered vertically

Instance Variables isCentered:

isCentered - true if actual text editor field should be centered vertically, false otherwise
