Overview

CPUWatcher implements a simple runaway process monitoring tool that will suspend a process that is taking up too much of Pharo's time and allow user interaction. By default it watches for a Process that is taking more than 80% of the time; this threshold can be changed.

CPUWatcher can also be used to show cpu percentages for each process from within the ProcessBrowser.
