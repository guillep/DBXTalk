Overview

A BitBltDisplayScanner displays characters on Screen or other Form with help of a BitBlt.

Instance Variables bitBlt: fillBlt:

bitBlt - the object which knows how to copy bits from one Form (the font glyph data) to another (the destination Form)

fillBlt - another object for copying form bits, initialized for displaying the background.
