Welcome to DBXTalk

DBXTalk is the relational database suite for the <a href="http://pharo.org">Pharo</a> live programming language. DBXTalk is an umbrella for the following projects:</p>

  • The DBXTalk database driver suite.
  • A SQL builder.
  • A database meta-data access.
  • The Glorp Object-Relational-Mapper for Pharo.
